Operators
---

- “Magmatic”:
  * `+`

- “Group-ish”:
  * `-` (both unary and binary)

- “Ring-ish”:
  * `*`

- “Euclidean ring-ish”:
  * `%`

- “Field-ic”:
  * `/`

- “Real-ish”:
  * `^`

- “String-ish”:
  * `~~`: concatenation

- “Relative”:

  * `==`
  * `!=`
  * `<`
  * `>`
  * `<=`
  * `>=`

- “Logical”:

  * `&&`
  * `||`
  * `!`

```

    1   2   3

> [1; 2,3]

```
